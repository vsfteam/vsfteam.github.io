Note
IMPORTENT For anything you want to wait, which is coming from vsf system, you can use vsf_task_wait_until() optionally. Those includues: VSF_EVT_XXXXX, sempahore, mutex and etc.

For VSF_EVT_XXXX, please use: vsf_task_wait_until( on_vsf_task_evt(VSF_EVT_XXXXX) ) ;

  For sempahore, please use:
  vsf_task_wait_until( vsf_sem_pend(...) ) ;


  For time, please use:
  vsf_task_wait_until( vsf_delay(...) ) ;
