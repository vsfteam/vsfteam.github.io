Detailed Description

Behaviour applied to the counter when the alarm matches.

  • If auto_reload_on_alarm is true, the counter is reloaded to reload_count and the alarm keeps firing periodically.
  • Otherwise the counter keeps running (if at all) and the user is responsible for re-arming via gptimer_set_alarm_action() again.
