initialize a mmc instance.

Parameters
[in]mmc_ptra pointer to structure vsf_mmc_t
[in]cfg_ptra pointer to structure vsf_mmc_cfg_t
Returns
vsf_err_t: VSF_ERR_NONE if mmc was initialized, or a negative error code
Note
