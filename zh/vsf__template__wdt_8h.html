◆ vsf_wdt_init()

vsf_err_t vsf_wdt_init ( vsf_wdt_t *  wdt_ptr,
vsf_wdt_cfg_t *  cfg_ptr 
)

初始化一个 wdt 实例

参数
[in]wdt_ptr结构体 vsf_wdt_t 的指针，参考 vsf_wdt_t
[in]cfg_ptr结构体 vsf_wdt_cfg_t 的指针，参考 vsf_wdt_cfg_t
返回
vsf_err_t: 如果 wdt 初始化成功返回 VSF_ERR_NONE , 失败返回负数。
注解
失败后不需要调用 vsf_wdt_fini() 反初始化。 vsf_wdt_init() 应该在除 vsf_wdt_capability() 之外的其他 WDT API 之前调用。
