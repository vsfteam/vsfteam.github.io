vsf_sdio_cfg_t *  cfg_ptr 
)

初始化一个 sdio 实例

参数
[in]sdio_ptr指向结构体 vsf_sdio_t 的指针
[in]cfg_ptr指向结构体 vsf_sdio_cfg_t 的指针
返回
vsf_err_t: 如果 sdio 初始化成功返回 VSF_ERR_NONE , 失败返回负数
注解
失败后不需要调用 vsf_sdio_fini() 反初始化。 vsf_sdio_init() 应该在除 vsf_sdio_capability() 之外的其他 sdio API 之前调用。

vsf_err_t vsf_sdio_get_configuration ( vsf_sdio_t *  sdio_ptr,
vsf_sdio_cfg_t *  cfg_ptr 
)

获取 SDIO 实例的当前配置

参数
[in]sdio_ptr指向结构体 vsf_sdio_t 的指针
[out]cfg_ptr指向配置结构体 vsf_sdio_cfg_t 的指针，用于存储当前配置
