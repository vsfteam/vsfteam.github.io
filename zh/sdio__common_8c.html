◆ vsf_sdio_init()

vsf_err_t vsf_sdio_init ( vsf_sdio_t *  sdio_ptr,
vsf_sdio_cfg_t *  cfg_ptr 
)

初始化一个 sdio 实例

参数
[in]sdio_ptr结构体 vsf_sdio_t 的指针，参考 vsf_sdio_t
[in]cfg_ptr结构体 vsf_sdio_cfg_t 的指针，参考 vsf_sdio_cfg_t
返回
vsf_err_t: 如果 sdio 初始化成功返回 VSF_ERR_NONE , 失败返回负数。
注解
失败后不需要调用 vsf_sdio_fini() 反初始化。 vsf_sdio_init() 应该在除 vsf_sdio_capability() 之外的其他 sdio API 之前调用。
