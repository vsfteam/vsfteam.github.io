◆ vsf_dma_init()

vsf_err_t vsf_dma_init ( vsf_dma_t *  dma_ptr)

初始化一个 dma 实例

参数
[in]dma_ptr结构体 vsf_dma_t 的指针，参考 vsf_dma_t
返回
vsf_err_t: 如果 dma 初始化成功返回 VSF_ERR_NONE , 失败返回负数。
注解
失败后不需要调用 vsf_dma_fini() 反初始化。 vsf_dma_init() 应该在除 vsf_dma_capability() 之外的其他 DMA API 之前调用。
