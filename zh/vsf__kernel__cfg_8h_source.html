192 should not happen."

194 # define __VSF_OS_SWI_NUM (VSF_OS_CFG_PRIORITY_NUM - 1)

200 calculated from macro VSF_OS_CFG_PRIORITY_NUM. Please define \
201 VSF_OS_CFG_PRIORITY_NUM in your vsf_usr_cfg.h (or any configuration header file \
202 included by vsf_usr_cfg.h)"
