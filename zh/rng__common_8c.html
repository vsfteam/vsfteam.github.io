vsf_err_t vsf_rng_init ( vsf_rng_t *  rng_ptr)

初始化一个 RNG 实例

参数
[in]rng_ptr指向结构体 vsf_rng_t 的指针
返回
vsf_err_t: 如果初始化成功返回 VSF_ERR_NONE，否则返回错误码
注解
失败后不需要调用 vsf_rng_fini() 进行反初始化。 vsf_rng_init() 应该在除 vsf_rng_capability() 之外的其他 RNG API 之前调用。
