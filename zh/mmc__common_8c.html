◆ vsf_mmc_init()

vsf_err_t vsf_mmc_init ( vsf_mmc_t * mmc_ptr,
vsf_mmc_cfg_t * cfg_ptr )

初始化一个 mmc 实例

参数
[in]mmc_ptr结构体 vsf_mmc_t 的指针，参考 vsf_mmc_t
[in]cfg_ptr结构体 vsf_mmc_cfg_t 的指针，参考 vsf_mmc_cfg_t
返回
vsf_err_t: 如果 mmc 初始化成功返回 VSF_ERR_NONE , 失败返回负数。
注解
失败后不需要调用 vsf_mmc_fini() 反初始化。 vsf_mmc_init() 应该在除 vsf_mmc_capability() 之外的其他 mmc API 之前调用。
