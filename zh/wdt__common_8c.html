vsf_err_t vsf_wdt_init ( vsf_wdt_t *  wdt_ptr,
vsf_wdt_cfg_t *  cfg_ptr 
)

初始化一个 WDT 实例

参数
[in]wdt_ptr：指向结构体vsf_wdt_t 的指针
[in]cfg_ptr：指向结构体vsf_wdt_cfg_t 的指针
返回
vsf_err_t：如果 WDT 初始化成功返回 VSF_ERR_NONE，失败返回负数
注解
初始化失败后不需要调用 vsf_wdt_fini() 进行反初始化。 必须在调用除 vsf_wdt_capability() 外的其他 WDT API 之前调用 vsf_wdt_init()。
