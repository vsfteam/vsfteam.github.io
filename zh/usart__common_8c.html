◆ vsf_usart_init()

vsf_err_t vsf_usart_init ( vsf_usart_t * usart_ptr,
vsf_usart_cfg_t * cfg_ptr )

初始化一个 usart 实例

参数
[in]usart_ptr结构体 vsf_usart_t 的指针，参考 vsf_usart_t
[in]cfg_ptr结构体 vsf_usart_cfg_t 的指针，参考 vsf_usart_cfg_t
返回
vsf_err_t: 如果 usart 初始化成功返回 VSF_ERR_NONE , 失败返回负数。
注解
失败后不需要调用 vsf_usart_fini() 反初始化。 vsf_usart_init() 应该在除 vsf_usart_capability() 之外的其他 usart API 之前调用。

◆ vsf_usart_txfifo_get_free_count()

uint_fast16_t vsf_usart_txfifo_get_free_count ( vsf_usart_t * usart_ptr)

获取 usart 发送队列的最大可发送数据的个数

参数
[in]usart_ptr结构体 vsf_usart_t 的指针，参考 vsf_usart_t
返回
uint_fast16_t: 返回当前 usart 发送队列的最大可发送数据的个数
注解
部分usart没有fifo，或者有fifo但是无法获取fifo的数量，可以返回1~2
  1. 数据寄存器为空，但是移位寄存器不为空
  2. 数据寄存器和移位寄存器都为空
