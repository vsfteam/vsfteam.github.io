◆ VSF_GPIO_CFG_PREFIX

#define VSF_GPIO_CFG_PREFIX   vsf

VSF_GPIO_CFG_PREFIX 是用来配置调用 vsf_gpio_*() 的时候实际调用的 API。 例如，当 VSF_GPIO_CFG_PREFIX 配置成 vsf_hw 的时候，调用 vsf_gpio_set_output() 实际调用的是 vsf_hw_gpio_set_output()

如果我们想要 xxxx.c 里调用 vsf_gpio_set_output() 实际调用的是 vsf_example_gpio_set_output()， 那么可以在 xxxx.c 源码前面配置：

#undef VSF_GPIO_CFG_PREFIX
#define VSF_GPIO_CFG_PREFIX vsf_example

预定义的 VSF GPIO 模式，可以在具体的 hal 驱动重新实现。

即使硬件不支持这些功能，但是这些模式是必须实现的。 如果硬件驱动里我们添加了新的模式，那么也需要定义宏 VSF_GPIO_MODE_MASK，它的值是所有模式的值的或。

  • INPUT/OUTPUT
    • VSF_GPIO_INPUT
    • VSF_GPIO_ANALOG
    • VSF_GPIO_OUTPUT_PUSH_PULL
    • VSF_GPIO_OUTPUT_OPEN_DRAIN
    • VSF_GPIO_EXTI
  • PULL_UP_AND_DOWN:
    • VSF_GPIO_NO_PULL_UP_DOWN
    • VSF_GPIO_PULL_UP
    • VSF_GPIO_PULL_DOWN

可选特性需要提供一个或者多个枚举选项，还需要提供同名的宏，方便用户在运行时判断是否支持。 如果它特性支持多个选项，建议提供对应的 MASK 选项，方便用户在运行时切换到不同的模式。

vsf_gpio_cfg_t cfg = {
.mode = VSF_GPIO_OUTPUT_PUSH_PULL,
...
};
vsf_gpio_pin_mask pin_mask = 0x00000001;
vsf_gpio_port_config_pins(&vsf_hw_gpi0, pin_mask, &cfg);

enum vsf_gpio_mode_t

预定义的 VSF GPIO 模式，可以在具体的 hal 驱动重新实现。

即使硬件不支持这些功能，但是这些模式是必须实现的。 如果硬件驱动里我们添加了新的模式，那么也需要定义宏 VSF_GPIO_MODE_MASK，它的值是所有模式的值的或。

  • INPUT/OUTPUT
    • VSF_GPIO_INPUT
    • VSF_GPIO_ANALOG
    • VSF_GPIO_OUTPUT_PUSH_PULL
    • VSF_GPIO_OUTPUT_OPEN_DRAIN
    • VSF_GPIO_EXTI
  • PULL_UP_AND_DOWN:
    • VSF_GPIO_NO_PULL_UP_DOWN
    • VSF_GPIO_PULL_UP
    • VSF_GPIO_PULL_DOWN

可选特性需要提供一个或者多个枚举选项，还需要提供同名的宏，方便用户在运行时判断是否支持。 如果它特性支持多个选项，建议提供对应的 MASK 选项，方便用户在运行时切换到不同的模式。

vsf_gpio_cfg_t cfg = {
.mode = VSF_GPIO_OUTPUT_PUSH_PULL,
...

枚举值
VSF_GPIO_INPUT 

将 GPIO 设置为通用输入模式。 选择 GPIO 作为替代功能。 与上拉、下拉和浮空配置无关。

VSF_GPIO_ANALOG 

将 GPIO 设置为模拟 IO（输入或输出）。 选择模拟或 GPIO 作为替代功能并设置为输入方向。

VSF_GPIO_OUTPUT_PUSH_PULL 

将 GPIO 设置为通用推挽输出模式。 选择 GPIO 作为替代功能。

VSF_GPIO_OUTPUT_OPEN_DRAIN 

将 GPIO 设置为通用开漏输出模式。 选择 GPIO 作为替代功能。

VSF_GPIO_EXTI 

将 GPIO 设置为外部中断 IO。 配置引脚以检测信号变化并生成中断。

VSF_GPIO_NO_PULL_UP_DOWN 

将 GPIO 设置为替代功能模式。 为外设特定功能（如 UART、SPI 等）配置引脚。 硬件可以根据硬件支持的情况采用以下方式之一实现：

  1. 仅实现 VSF_GPIO_AF
  2. 实现 VSF_GPIO_AF_OUTPUT_PUSH_PULL 和 VSF_GPIO_AF_OUTPUT_OPEN_DRAIN
  3. 实现 VSF_GPIO_AF_OUTPUT_PUSH_PULL、VSF_GPIO_AF_OUTPUT_OPEN_DRAIN 和 VSF_GPIO_AF_INPUT

不启用上拉或下拉电阻。 与 IO 方向和替代功能无关。

VSF_GPIO_PULL_UP 

启用内部上拉电阻。 与 IO 方向和替代功能无关。

VSF_GPIO_PULL_DOWN 

启用内部下拉电阻。 与 IO 方向和替代功能无关。

VSF_GPIO_EXTI_MODE_NONE 

不启用外部中断检测。

VSF_GPIO_EXTI_MODE_LOW_LEVEL 

在低电平信号上检测中断。

VSF_GPIO_EXTI_MODE_HIGH_LEVEL 

在高电平信号上检测中断。

VSF_GPIO_EXTI_MODE_RISING 

在信号上升沿检测中断。

VSF_GPIO_EXTI_MODE_FALLING 

在信号下降沿检测中断。

VSF_GPIO_EXTI_MODE_RISING_FALLING 

在信号上升沿和下降沿检测中断。
