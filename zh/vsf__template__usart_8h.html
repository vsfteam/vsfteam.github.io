注解
uart_status_t should provide dedicated bits for ! indicating whether a read or write timeout event is detected ! or not: ! bIsRXTimeOut ! bIsTXTimeOut ! !
uart_status_t should provide dedicated bits for indicating ! whether a read or write operation is cancelled by user: ! bIsRXCancelled ! bIsTXCancelled ! !
uart_status_t should provide dedicated bits for indicating ! whether a read or write operation is encountered an error: ! bIsRXErrorDetected ! bIsTXErrorDetected ! !
: ! Those bits will not be cleared until corresponding transmission ! operation is request. E.g. When a Block.Read.Request is called, ! then the bIsRXTimeOut bit should be cleared.
