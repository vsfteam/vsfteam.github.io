vsf_err_t vsf_rtc_init ( vsf_rtc_t * rtc_ptr,
vsf_rtc_cfg_t * cfg_ptr )
extern

初始化一个 rtc 实例

参数
[in]rtc_ptr结构体 vsf_rtc_t 的指针，参考 vsf_rtc_t
[in]cfg_ptr结构体 vsf_rtc_cfg_t 的指针，参考 vsf_rtc_cfg_t
返回
vsf_err_t: 如果 rtc 初始化成功返回 VSF_ERR_NONE , 失败返回负数。
注解
失败后不需要调用 vsf_rtc_fini() 反初始化。 vsf_rtc_init() 应该在除 vsf_rtc_capability() 之外的其他 rtc API 之前调用。
