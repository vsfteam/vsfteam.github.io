◆ vsf_timer_init()

vsf_err_t vsf_timer_init ( vsf_timer_t *  timer_ptr,
vsf_timer_cfg_t *  cfg_ptr 
)
extern

初始化定时器实例

参数
[in]timer_ptr指向结构体 vsf_timer_t 的指针
[in]cfg_ptr指向配置结构体 vsf_timer_cfg_t 的指针
返回
vsf_err_t: 如果初始化成功返回 VSF_ERR_NONE，否则返回错误码
注解
失败后不需要调用 vsf_timer_fini() 进行反初始化。 vsf_timer_init() 应该在除 vsf_timer_capability() 之外的其他 Timer API 之前调用。
