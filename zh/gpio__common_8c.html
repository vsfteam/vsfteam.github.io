配置 gpio 实例的一个或者多个引脚

参数
[in]gpio_ptr结构体 vsf_gpio_t 的指针，参考 vsf_gpio_t
[in]pin_mask引脚掩码，每一个引脚对应一个位，该 bit 的值位 1 表示需要配置
[in]cfg_ptr结构体 vsf_gpio_cfg_t 的指针，参考 vsf_gpio_cfg_t
返回
vsf_err_t: 如果 GPIO 配置成功返回 VSF_ERR_NONE，失败返回负数。

uint16_t  pin_index,
vsf_gpio_cfg_t *  cfg_ptr 
)

获取 gpio 实例的指定引脚的配置

参数
[in]gpio_ptr结构体 vsf_gpio_t 的指针，参考 vsf_gpio_t
[in]pin_index引脚索引（从0开始），指定要获取配置的引脚
[out]cfg_ptr结构体 vsf_gpio_cfg_t 的指针，用于存储配置，参考 vsf_gpio_cfg_t
返回
vsf_err_t: 如果 GPIO 引脚配置获取成功返回 VSF_ERR_NONE，失败返回负数。
注解
此函数仅获取单个引脚的配置。要获取多个引脚的配置，请为每个引脚单独调用此函数。

◆ vsf_gpio_read()

vsf_gpio_pin_mask_t vsf_gpio_read ( vsf_gpio_t *  gpio_ptr)

读取 gpio 实例的所有引脚的值

参数
[in]gpio_ptr指向结构体 vsf_gpio_t 的指针
返回
vsf_gpio_pin_mask_t: 所有引脚的值，1 表示高电平，0 表示低电平
注解
此函数读取引脚的实际电平值（输入值）。对于配置为输出的引脚，如果引脚被外部驱动 或存在硬件问题，此值可能与输出寄存器值不同。

◆ vsf_gpio_read_output_register()

vsf_gpio_pin_mask_t vsf_gpio_read_output_register ( vsf_gpio_t *  gpio_ptr)

读取 gpio 实例的所有引脚的输出寄存器值

参数
[in]gpio_ptr指向结构体 vsf_gpio_t 的指针
返回
vsf_gpio_pin_mask_t: 所有引脚的输出寄存器值，1 表示高电平，0 表示低电平
注解
此函数读取输出寄存器值，表示 GPIO 控制器试图在引脚上驱动的值。这与 vsf_gpio_read() 不同，后者读取引脚的实际电平值。对于输出引脚，此函数显示预期的输出状态， 不受外部条件影响。

vsf_gpio_pin_mask_t  pin_mask 
)

使能指定的一个或者多个引脚的中断

参数
[in]gpio_ptr指向结构体 vsf_gpio_t 的指针
[in]pin_mask引脚掩码，每一个引脚对应一个位，1 表示该位需要使能，0 表示该位不需要使能
返回
vsf_err_t: 如果成功返回 VSF_ERR_NONE，失败返回负数
注解
对于一些芯片，中断优先级可能是 gpio 上所有引脚共用的。
在中断使能之前，应该清除所有悬挂的中断。
