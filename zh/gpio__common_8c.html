◆ vsf_gpio_set_direction()

void vsf_gpio_set_direction ( vsf_gpio_t *  gpio_ptr,
uint32_t  direction_mask,
uint32_t  pin_mask 
)

set the specified pins to corresponding directions ! pseudo code: ! uint32_t temp = VSF_GPIOx.DIR; ! temp &= ~pin_mask; ! VSF_GPIOx.DIR = temp | (direction_bitmap & pin_mask); ! !

参数
gpio_ptrthe address of target vsf_gpio_t object (given port) !
direction_bitmapdirection bitmap which will be ANDDed with pin_mask ! direction value is 1 for output ! direction value is 0 for input !
pin_maskpin mask to mark the target pin within a given port !
返回
none
