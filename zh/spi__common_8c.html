◆ vsf_spi_fifo_transfer()

void vsf_spi_fifo_transfer ( vsf_spi_t *  spi_ptr,
void *  out_buffer_ptr,
uint_fast32_t *  out_offset_ptr,
void *  in_buffer_ptr,
uint_fast32_t *  in_offset_ptr,
uint_fast32_t  count 
)

在 SPI 实例上执行基于 FIFO 的数据传输

参数
[in,out]spi_ptr指向结构体 vsf_spi_t 的指针
[in]out_buffer_ptr指向发送数据缓冲区的指针（仅接收时可为 NULL）
[in,out]out_offset_ptr指向发送缓冲区偏移的指针
[out]in_buffer_ptr指向接收数据缓冲区的指针（仅发送时可为 NULL）
[in,out]in_offset_ptr指向接收缓冲区偏移的指针
[in]count要传输的数据单元数
注解
数据单元大小由 SPI 模式中配置的数据大小决定
在主机模式下，out_buffer_ptr 数据通过 MOSI 引脚发送，in_buffer_ptr 从 MISO 引脚接收
在从机模式下，out_buffer_ptr 数据通过 MISO 引脚发送，in_buffer_ptr 从 MOSI 引脚接收

◆ vsf_spi_request_transfer()

vsf_err_t vsf_spi_request_transfer ( vsf_spi_t *  spi_ptr,
void *  out_buffer_ptr,
void *  in_buffer_ptr,
uint_fast32_t  count 
)

请求 SPI 数据传输操作

参数
[in,out]spi_ptr指向 SPI 实例结构体 vsf_spi_t 的指针
[in]out_buffer_ptr指向输出数据缓冲区的指针（仅接收时为 NULL）
[in]in_buffer_ptr指向输入数据缓冲区的指针（仅发送时为 NULL）
[in]count要传输的数据单元数量
返回
vsf_err_t: 如果传输成功启动则返回 VSF_ERR_NONE，否则返回错误码
注解
这是一个异步操作，注册完成中断以获得通知
如果 out_buffer_ptr 和 in_buffer_ptr 都不为 NULL，则以全双工模式交换数据
在主机模式下，out_buffer_ptr 数据通过 MOSI 引脚发送，in_buffer_ptr 从 MISO 引脚接收
在从机模式下，out_buffer_ptr 数据通过 MISO 引脚发送，in_buffer_ptr 从 MOSI 引脚接收

获取当前或最后一次 SPI 操作中已传输的数据单元数
