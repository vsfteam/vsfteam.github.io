◆ vsf_flash_init()

vsf_err_t vsf_flash_init ( vsf_flash_t *  flash_ptr,
vsf_flash_cfg_t *  cfg_ptr 
)

初始化一个 FLASH 实例

参数
[in]flash_ptr指向结构体 vsf_flash_t 的指针
[in]cfg_ptr指向配置结构体 vsf_flash_cfg_t 的指针
返回
vsf_err_t: 如果初始化成功返回 VSF_ERR_NONE，否则返回错误码
注解
初始化失败后不需要调用 vsf_flash_fini() 进行反初始化。
vsf_flash_init() 应该在除 vsf_flash_capability() 之外的其他 FLASH API 之前调用。
