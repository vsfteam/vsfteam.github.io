使用默认值初始化 vsf_dma_channel_hint_t 结构体

使用默认值初始化 vsf_dma_channel_hint_t 结构体

参数
[in]__CHANNEL通道号（使用负值表示自动分配）
[in]__REQUEST_LINE外设请求线编号
[in]...可选的中断优先级。如果不提供，默认为 vsf_arch_prio_invalid。
注解
使用此宏确保正确初始化，特别是对于 vsf_arch_prio_t， 其无效值可能不是 0（vsf_arch_prio_invalid = -1）。
如果未指定优先级，将默认为 vsf_arch_prio_invalid。

// 自动通道分配，不指定优先级 vsf_dma_channel_hint_t hint1 = VSF_DMA_CHANNEL_HINT_INIT(-1, 5);

// 指定通道号，不指定优先级 vsf_dma_channel_hint_t hint2 = VSF_DMA_CHANNEL_HINT_INIT(3, 10);

// 自动通道分配，指定优先级 vsf_arch_prio_t test_prio = 2; vsf_dma_channel_hint_t hint3 = VSF_DMA_CHANNEL_HINT_INIT(-1, 7, test_prio);

// 指定通道号，指定优先级 vsf_dma_channel_hint_t hint4 = VSF_DMA_CHANNEL_HINT_INIT(5, 12, 1);
