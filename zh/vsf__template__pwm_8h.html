vsf_pwm_cfg_t *  cfg_ptr 
)
extern

初始化一个 pwm 实例

参数
[in]pwm_ptr结构体 vsf_pwm_t 的指针，参考 vsf_pwm_t
[in]cfg_ptr结构体 vsf_pwm_cfg_t 的指针，参考 vsf_pwm_cfg_t
返回
vsf_err_t: 如果 pwm 初始化成功返回 VSF_ERR_NONE , 失败返回负数。
注解
失败后不需要调用 vsf_pwm_fini() 反初始化。 vsf_pwm_init() 应该在除 vsf_pwm_capability() 之外的其他 pwm API 之前调用。
