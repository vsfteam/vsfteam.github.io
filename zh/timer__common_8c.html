◆ vsf_timer_init()

vsf_err_t vsf_timer_init ( vsf_timer_t *  timer_ptr,
vsf_timer_cfg_t *  cfg_ptr 
)

初始化一个 timer 实例

参数
[in]timer_ptr结构体 vsf_timer_t 的指针，参考 vsf_timer_t
[in]cfg_ptr结构体 vsf_timer_cfg_t 的指针，参考 vsf_timer_cfg_t
返回
vsf_err_t: 如果 timer 初始化成功返回 VSF_ERR_NONE , 失败返回负数。
注解
失败后不需要调用 vsf_timer_fini() 反初始化。 vsf_timer_init() 应该在除 vsf_timer_capability() 之外的其他 timer API 之前调用。
