返回
fsm_rt_t: 如果 I2C 实例已禁用返回 fsm_rt_cpl，如果 I2C 实例正在禁用过程中返回 fsm_rt_on_going

◆ vsf_i2c_master_fifo_transfer()

fsm_rt_t vsf_i2c_master_fifo_transfer ( vsf_i2c_t *  i2c_ptr,
uint16_t  address,
vsf_i2c_cmd_t  cmd,
uint_fast16_t  count,
uint8_t *  buffer_ptr,
vsf_i2c_cmd_t *  cur_cmd_ptr,
uint_fast16_t *  offset_ptr 
)

I2C 主机进行一次 FIFO 传输

参数
[in,out]i2c_ptr结构体 vsf_i2c_t 的指针
[in]addressI2C 传输的地址
[in]cmdI2C 命令
[in]count要传输的数据数量
[in,out]buffer_ptrI2C 传输缓冲区（当 count 不为 0 时，不能为空指针）
[in,out]cur_cmd_ptr当前 I2C 命令指针（第一次调用前必须清零）
[in,out]offset_ptr当前偏移指针（第一次调用前必须清零）
返回
fsm_rt_t: 传输状态
  • fsm_rt_cpl: 传输成功完成
  • fsm_rt_err: 主机写操作收到 NAK (Not Acknowledge)
  • fsm_rt_on_going: 传输进行中，需要继续调用
注解
当 count 不为 0 时，buffer_ptr 不能为空指针。 cur_cmd_ptr 和 offset_ptr 指向的变量在第一次调用之前必须清零。 当返回 fsm_rt_on_going 时，需要继续调用此函数直到完成。
