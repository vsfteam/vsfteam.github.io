◆ vsf_i2c_init()

vsf_err_t vsf_i2c_init ( vsf_i2c_t *  i2c_ptr,
vsf_i2c_cfg_t *  cfg_ptr 
)

初始化一个 i2c 实例

参数
[in]i2c_ptr结构体 vsf_i2c_t 的指针，参考 vsf_i2c_t
[in]cfg_ptr结构体 vsf_i2c_cfg_t 的指针，参考 vsf_i2c_cfg_t
返回
vsf_err_t: 如果 i2c 初始化完成返回 VSF_ERR_NONE , 否则返回负数。
注解
初始化一个i2c实例应该包括开启对应的电源门控，不包括引脚复用功能

◆ vsf_i2c_enable()

fsm_rt_t vsf_i2c_enable ( vsf_i2c_t *  i2c_ptr)

使能 i2c 实例

参数
[in]i2c_ptr结构体 vsf_i2c_t 的指针，参考 vsf_i2c_t
返回
fsm_rt_t: 如果使能成功，返回 fsm_rt_cpl, 未完成初始化返回 fsm_rt_onging
注解
vsf_i2c_enable() 应该在 vsf_i2c_init() 之后调用。时钟门控位只需要确保在 vsf_i2c_enable() 之后开启。

◆ vsf_i2c_disable()

fsm_rt_t vsf_i2c_disable ( vsf_i2c_t *  i2c_ptr)

禁能 i2c 实例

参数
[in]i2c_ptr结构体 vsf_i2c_t 的指针，参考 vsf_i2c_t
返回
fsm_rt_t: 如果禁能成功，返回 fsm_rt_cpl, 未完成初始化返回 fsm_rt_onging
注解
需要确保在 vsf_i2c_disable() 之后调用 vsf_i2c_enable() 是可以正常工作的， 而不需要调用 vsf_i2c_init()
