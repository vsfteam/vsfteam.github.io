初始化一个 PWM 实例

参数
[in]pwm_ptr指向结构体 vsf_pwm_t 的指针
[in]cfg_ptr指向配置结构体 vsf_pwm_cfg_t 的指针
返回
vsf_err_t: 如果初始化成功返回 VSF_ERR_NONE，否则返回错误码
注解
失败后不需要调用 vsf_pwm_fini() 进行反初始化。 vsf_pwm_init() 应该在除 vsf_pwm_capability() 之外的其他 PWM API 之前调用。

vsf_pwm_cfg_t *  cfg_ptr 
)

获取 PWM 实例的当前配置

参数
