initialize a wdt instance.

Parameters
[in]wdt_ptra pointer to structure vsf_wdt_t
[in]cfg_ptra pointer to structure vsf_wdt_cfg_t
Returns
vsf_err_t: VSF_ERR_NONE if wdt was initialized, or a negative error code
Note
