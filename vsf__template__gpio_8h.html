◆ VSF_GPIO_CFG_PREFIX

#define VSF_GPIO_CFG_PREFIX   vsf

VSF_GPIO_CFG_PREFIX is used to set the actual API call when calling the vsf_gpio_*(). For example, if you configure VSF_GPIO_CFG_PREFIX to be vsf_hw, then call vsf_gpio_init(), which is actually vsf_hw_gpio_init()

If we want the call to vsf_gpio_init in xxxx.c to actually call vsf_extern_gpio_init, then it can be configured in front of the .c:

#undef VSF_GPIO_CFG_PREFIX
#define VSF_GPIO_CFG_PREFIX vsf_extern
